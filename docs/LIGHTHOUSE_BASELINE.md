# Lighthouse Baseline

**Target (per architecture):** Overall score **90+**; **LCP &lt; 2.5s**; CLS &lt; 0.1; INP &lt; 200ms.

Run monthly (e.g. Chrome DevTools Lighthouse or PageSpeed Insights). Document results here or in Google Search Console. Speed over beauty — every extra second costs conversions.

## How to run

1. Open homepage (and key pages) in Chrome.
2. DevTools → Lighthouse tab → Analyze page load.
3. Record Performance, LCP, CLS, and any INP/TBT notes.
4. Fix regressions (e.g. large images, render-blocking resources).
